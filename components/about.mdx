## Overview

The About component presents professional information in a two-column layout: a biographical section with tech stack on the left, and a timeline of professional milestones on the right. It features interactive hover effects and a modern card-based design.

  <div class="absolute top-0 right-0 w-1/3 h-full bg-linear-to-l from-primary/5 to-transparent pointer-events-none">

      <!-- Bio and Tech Stack -->
      <!-- Timeline -->
    </div>
  </div>

## Interactive Effects

### Hover Transitions

Timeline icons feature sophisticated hover effects:

```css
group-hover:bg-primary: Icon background changes to primary color
group-hover:text-ocean-dark: Text inverts to dark
group-hover:shadow-[0_0_15px_rgba(204,204,254,0.5)]: Adds glow
transition-all: Smooth animation for all properties
```

### Tech Badge Interactions

Each technology badge responds to hover:

1. Border color changes from `border-primary/40` to `border-primary`
2. Glow effect appears: `shadow-[0_0_15px_rgba(204,204,254,0.3)]`
3. All transitions are smooth via `transition-all`

## Background Effects

Subtle gradient overlay adds depth:

```html
<div class="absolute top-0 right-0 w-1/3 h-full bg-linear-to-l from-primary/5 to-transparent pointer-events-none">
</div>
```

<Note>
The `pointer-events-none` class ensures background elements don't interfere with user interactions.
</Note>

## Styling Details

### Typography
- **Headings:** `font-serif` for elegance
- **Body Text:** `font-light` for readability
- **Labels:** `font-bold uppercase tracking-wider` for emphasis

### Color System
- **Primary Text:** `text-white` for main content
- **Secondary Text:** `text-slate-400` for descriptions
- **Accent:** `text-primary` for highlights and icons
- **Background:** `bg-ocean-dark` and `bg-ocean-card` for depth

### Border and Shadow
- Borders use opacity variants: `border-primary/40`, `border-white/5`
- Custom shadows create glow effects: `shadow-[0_0_10px_rgba(204,204,254,0.4)]`
- Backdrop blur for glassmorphism: `backdrop-blur-sm`
