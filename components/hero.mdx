## Overview

The Hero component serves as the landing section of the portfolio, creating an impactful first impression with animated elements, gradient backgrounds, and responsive layouts. It features a two-column grid layout with introduction text on one side and an animated profile image on the other.

## Animated Profile Section

The profile image features multiple animated orbital rings:

```html
<div class="relative w-64 h-64 md:w-80 md:h-80 lg:w-112.5 lg:h-112.5">
  <!-- Outer rotating ring -->
  <div class="absolute inset-0 rounded-full border border-primary/20 scale-110 animate-[spin_10s_linear_infinite] shadow-[0_0_30px_rgba(204,204,254,0.1)]">
  </div>
  
  <!-- Reverse rotating dashed ring -->
  <div class="absolute inset-0 rounded-full border border-dashed border-primary/30 scale-125 animate-[spin_20s_linear_infinite_reverse]">
  </div>
  
  <!-- Profile image container -->
  <div class="relative w-full h-full rounded-full border-4 border-primary/50 shadow-[0_0_50px_rgba(204,204,254,0.25)] overflow-hidden bg-ocean-card">
    <img
      alt="Jamir Garro Portrait"
      class="w-full h-full object-cover hover:scale-105 transition-transform duration-700"

    />

```

<Note>
**Animation Details:**
- First ring: 10-second rotation (`spin_10s_linear_infinite`)
- Second ring: 20-second reverse rotation (`spin_20s_linear_infinite_reverse`)
- Image scales up on hover for interactive feedback
</Note>

## Styling Approach

### Color Palette
- **Background:** `bg-ocean-dark` - Deep navy background
- **Primary Accent:** `bg-primary` (#CCCCFE) - Lavender blue
- **Text:** `text-white`, `text-slate-300` - White and light gray

### Responsive Design

<Tabs>
  <Tab title="Mobile">
    - Single column layout
    - Centered text alignment
    - Stacked buttons
    - Image displayed first (order-1)
  </Tab>
  
  <Tab title="Desktop">
    - Two-column grid (`lg:grid-cols-2`)
    - Left-aligned text
    - Side-by-side buttons
    - Image on the right (order-2)
  </Tab>
</Tabs>

## Interactive Features

1. **Hover Effects:** Buttons lift and glow on hover
2. **Animations:** Multiple elements use CSS animations (pulse, spin, bounce)
3. **Smooth Transitions:** All interactive elements have `transition-all` applied
4. **Group Interactions:** Icon animations triggered by parent hover state

<Note>
The component uses Material Symbols icons loaded globally. Ensure the icon font is included in your Layout component.
</Note>
