## Overview

The Contact component provides a fully functional contact form with client-side JavaScript handling, form validation, and success/error messaging. It integrates with Formspree for form submission and includes social media links with hover effects.

## Form Handling Script

Client-side JavaScript manages form submission:

<CodeGroup>
```javascript Form Submission Handler
const form = document.getElementById("my-form");

async function handleSubmit(event) {
  event.preventDefault();
  const status = document.getElementById("my-form-status");
  const data = new FormData(event.target);
  
  fetch(event.target.action, {
    method: form.method,
    body: data,
    headers: {
      Accept: "application/json",
    },
  })
    .then((response) => {
      if (response.ok) {
        status.classList.add("opacity-100", "-translate-y-2");
        status.innerHTML = "Thanks for your submission!";
        setTimeout(() => {
          status.innerHTML = "";
        }, 2000);
        form.reset();
      } else {
        // Error handling
      }
    })
    .catch((error) => {
      status.innerHTML = "Oops! There was a problem submitting your form";
    });
}

form.addEventListener("submit", handleSubmit);
```

```markdown Key Features
- Prevents default form submission
- Uses Fetch API for AJAX submission
- Shows success message with animation
- Resets form after successful submission
- Handles errors gracefully
- Auto-hides success message after 2 seconds
```
</CodeGroup>
