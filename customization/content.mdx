# Content Customization

This guide covers how to update all the personal content in your portfolio, from hero text to project showcases.

## Personal Information

### Hero Section

Update your main introduction in `/src/components/Hero.astro`:

**Location:** `/src/components/Hero.astro:30-44`

```astro
<h1 class="font-display font-black text-white text-4xl md:text-5xl lg:text-6xl mb-6">
  Junior Frontend Developer
  <span class="text-gradient">
    / <br class="hidden lg:block" />Jamir Garro
  </span>
</h1>
<p class="text-slate-300 text-lg md:text-xl max-w-2xl mx-auto lg:mx-0 mb-10">
  Specializing in creating responsive, high-performance interfaces with
  React, Astro, and Tailwind CSS. Focused on clean code and fluid user
  experiences.
</p>
```

**What to change:**
- Job title (e.g., "Junior Frontend Developer")
- Your name in the `<span class="text-gradient">` element
- Description text about your skills and focus areas

### Availability Badge

Update your work status in the hero section:

**Location:** `/src/components/Hero.astro:22-29`

```astro
<div class="inline-flex items-center gap-2 px-3 py-1 mb-6 rounded-full border border-primary/30 bg-primary/5 text-primary text-xs font-bold uppercase tracking-widest">
  <span class="w-2 h-2 rounded-full bg-primary animate-pulse"></span>
  Available for work
</div>
```

Change "Available for work" to reflect your current status (e.g., "Open to opportunities", "Freelancing", etc.).

### Hero Image

Replace the hero portrait image:

**Location:** `/src/components/Hero.astro:2-3`

```astro
import HeroImage from "../assets/images/hero-image.webp";
```

Replace `/src/assets/images/hero-image.webp` with your own portrait image. For best results:
- Use square dimensions (e.g., 800x800px)
- Save as WebP for optimal performance
- Ensure good contrast against dark backgrounds

## Contact Form

### Formspree Configuration

The contact form uses [Formspree](https://formspree.io/) for form handling.

**Location:** `/src/components/Contact.astro:22-26`

```astro
<form
  id="my-form"
  action="https://formspree.io/f/meeljvav"
  method="POST"
  class="grid grid-cols-1 md:grid-cols-2 gap-8"
>
```

**To configure your own form:**

1. Go to [formspree.io](https://formspree.io/) and create a free account
2. Create a new form and get your form endpoint
3. Replace `https://formspree.io/f/meeljvav` with your endpoint
4. The form will automatically start sending submissions to your email

### Form Fields

The form includes three fields:

- **Name** (required)
- **Email** (required)
- **Message** (required)

To add more fields, follow the existing pattern:

```astro
<div class="space-y-2">
  <label class="text-xs font-bold uppercase text-primary ml-1 tracking-wider">
    Your Label
  </label>
  <input
    class="w-full bg-ocean-card border-white/5 rounded-xl px-4 py-3 focus:ring-2 focus:ring-primary text-white"
    placeholder="Placeholder text"
    type="text"
    name="fieldName"
    required
  />
</div>
```

## Best Practices

1. **Keep content concise**: Aim for clarity and impact
2. **Update regularly**: Keep projects and experience current
3. **Optimize images**: Use WebP format and appropriate dimensions
4. **Test links**: Ensure all external links work correctly
5. **Accessibility**: Always include alt text for images
6. **Consistent tone**: Maintain a professional, approachable voice throughout
